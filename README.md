# CAM/3iAtlas

A conceptual framework exploring large-scale cosmic morphology and
observational reconstruction approaches applied to deep-field
astronomical imagery.

## Overview
CAM/3iAtlas proposes structured interpretation methods for analyzing
cosmic-scale visual structures and density formations observed in
astronomical deep-field imagery.

## Pre-Publish Bundle Support
This repository now supports one-step launch from either:
- `TOTALITY_RELEASE_PACKAGE.tar.gz` (+ optional `.sha256`), or
- `TOTALITY_PrePublish_Bundle.zip`

Run:

```bash
bash TOTALITY_ONE_STEP_DEPLOY.sh
```

For pre-publish bundles, deployment validates the expected structure
(`metadata.json`, `scripts/`, `supplementary/`, and thesis files)
and launches a detected publish script when available.
